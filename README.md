# A PHP7 V8js Docker image for Drupal
PHP-7 FPM image based on the **PHP** packages(**php-7.1.***).

This fork "https://github.com/madecomfy/php7-v8js" is advanced and improved to use Drupal.

## Main Built features
* Ansible
* Git
* MySQL-Client
* PHP-FPM
* PHP Memcached
* PHP Redis
* PHP PostgreSQL
* PHP MongoDB
* Xdebug
* Composer
* Drush
* POSIX
* Patch
* TAR
* Wget
* SSH
* Rsync
* Nano
* Halite
